# Veltrix - Team 48

Veltrix is an e-commerce web application that focuses on gaming-related products and services into a single digital hub. Gamers currently have to browse multiple retailers and websites to find consoles, accessories, merchandise, and digital gaming content, which is inefficient and time-consuming. Veltrix aims to address these issues by allowing users to browse gaming categories, perform searches, and view structured item information all in one place. The platform lies within the e-commerce and digital gaming service domain and is targeted at a wide audience: casual gamers, heavy gamers, streamers, early-stage influencers, as well as both children and adults interested in gaming content. By consolidating information into one environment, Veltrix enhances efficiency, reduces search friction, and improves the overall browsing experience.

Currently hosted on: https://cs2team48.cs2410-web01pvm.aston.ac.uk/
